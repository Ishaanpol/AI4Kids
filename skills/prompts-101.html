Prompts 101

Learn to give clear instructions to AI — like talking to a very literal robot.

How to Write a Good Prompt

  1. Be Specific: “Write a 2-sentence story about a kind robot.”
  2. Give Examples: “Here are two jokes I like… Make one similar.”
  3. Set Limits: “Use simple words. For ages 8–12.”

Try It

Ask an adult before using any AI tool. You can practice prompts on paper first!

Tip: Add one example to improve answers.

Mini Activity

On paper, write a prompt with: goal, details, limits. Example: "Explain gravity in 3 simple sentences for a 4th grader."
